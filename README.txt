== Description ==
Envía información de pediso de Wooommerce a QuieroMiFactura para generar CFDI.
Versión para CFDI 4

== Installation ==

1.- Desde WP Admin, en menú "Plugins" -> "Añadir Nuevo", click en el botón de "Subir Plugin"1, y seleccionar el archivo 'qmf4.zip'
2. Activar el plugin desde el menú "Plugin" de WP Admin
